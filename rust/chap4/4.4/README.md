# 4.4 재귀 록

## ch4_4_reent_c

`ch4_4_reent_c`는 C 언어이므로 `make`를 이용해 컴파일 한 뒤 실행합니다.

```sh
$ cd ch4_4_reent_c
$ make
$ ./reent
```

## ch4_4_reent_rust

`ch4_4_reent_rust`는 Rust 언어이므로 `cargo`를 이용해 컴파일 및 실행합니다.

> ⚠️ **경고**: 이 ㅗ드는 데드록에 빠져 프로그램이 정지하지 않으므로 ctrl + c로 실행을 정지하기 바랍니다.

```sh
$ cd ch4_4_reent_rust
$ cargo run --release
```
